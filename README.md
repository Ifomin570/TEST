**Правило №3:** И резюме, и портфолио должны выглядеть в едином стиле (цвета, шрифты, тональность).

**Правило №4:** Держите всё актуальным. Если проект умер (демо не работает) — уберите ссылку или почините.

**Чек-лист финальной проверки:**
- [ ] В резюме есть ссылка на GitHub/сайт-портфолио  
- [ ] В портфолио есть ссылка на резюме  
- [ ] Контакты везде одинаковые  
- [ ] Все ссылки работают (проверьте перед отправкой)
